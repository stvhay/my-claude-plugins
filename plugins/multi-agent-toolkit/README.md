# multi-agent-toolkit

Multi-agent coordination skills: council debates and parallel research.

## Installation

```bash
# From marketplace
/plugin marketplace add stvhay/my-claude-plugins
/plugin install multi-agent-toolkit@my-claude-plugins
```

## Skills

### council

Multi-agent debate system for decision-making. Specialized agents discuss topics in rounds, respond to each other's points, and surface insights through intellectual friction.

- **Debate** — Full 3-round structured debate with visible transcript
- **Quick** — Single-round fast perspective check

### research

Multi-source parallel research with URL verification protocol.

- **Quick** — 1 agent, ~10-15s
- **Standard** — 2 agents in parallel, ~15-30s (default)
- **Extensive** — 9 agents in parallel, ~60-90s
