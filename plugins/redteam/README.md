# redteam

Adversarial analysis with parallel agent deployment: stress-test ideas and produce content through competition.

## Installation

```bash
# From marketplace
/plugin marketplace add stvhay/my-claude-plugins
/plugin install redteam@my-claude-plugins
```

## Skills

| Skill | Purpose |
|-------|---------|
| redteam | Adversarial analysis using 32 parallel agents (ParallelAnalysis) or competing proposals (AdversarialValidation) |

## Workflows

- **ParallelAnalysis** — Break argument into 24 atomic claims, attack from 32 expert perspectives, synthesize into steelman + counter-argument
- **AdversarialValidation** — Competing proposals → brutal critique → collaborative synthesis
